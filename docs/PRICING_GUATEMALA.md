# Propuesta de precios (Guatemala)

Base de referencia para SaaS de lealtad en Guatemala y LATAM:

- Plan de entrada accesible para micro y pequenos negocios.
- Precio medio competitivo para negocios con varias sedes y automatizaciones.
- Plan alto para cadenas con integraciones y gobierno avanzado.

## Precios recomendados (sin IVA)

- EMPRENDEDOR: Q149/mes (Q1,490/año)
- NEGOCIO: Q399/mes (Q3,990/año)
- EMPRESA: Q999/mes (Q9,990/año)

## Mensajería incluida por plan

- EMPRENDEDOR: 250 mensajes/mes, excedente Q0.20/mensaje
- NEGOCIO: 1,000 mensajes/mes, excedente Q0.16/mensaje
- EMPRESA: 3,000 mensajes/mes, excedente Q0.12/mensaje

## Notas comerciales

- IVA Guatemala: 12% adicional sobre el precio publicado.
- Mensajeria (SMS/WhatsApp): cobrar por consumo para no perder margen.
- Descuento anual sugerido: ~2 meses gratis ya incluido en precio anual.
- Onboarding opcional (una sola vez): Q300-Q1,500 segun complejidad.

## Guardrails de margen

- Meta de margen bruto SaaS: >= 75%.
- Revisar trimestralmente costos variables (mensajeria, soporte, infraestructura).
- Si el costo por negocio supera 25% del ingreso mensual, ajustar precio o limites.
